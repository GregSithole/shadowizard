# Shadowizard

Based on the DesignCourse's shadowizard youtube video regarding creating your own NPM Package

## Installation

`npm i shadowizard-greg --save`

## Usage

```
import { shadowizard } from 'shadowizard-greg'

shadowizard({
    shadow_type: 'soft',
    padding: false
})
```

## Options 
* *shadow_type* - _hard | soft_ (defaults to soft)
* *padding* - _boolean_
